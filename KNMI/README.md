KNMI Data:
Beautifully rich, but not exactly user-friendly.
KNMI has one of the richest weather datasets in the world: daily measurements of precipitation, temperature, and wind, with weather data spanning many years. A goldmine for researchers and data enthusiasts.
But the API is something you would rather not work with.

The points:
stns=ALL sometimes works, sometimes doesn’t — without a clear error message.
Too much data? You get an HTML page back instead of an error code.
Rainfall stations are stored in ZIP files on a CDN, following a pattern that is not documented anywhere.
Limits? You discover them yourself through trial and error.
